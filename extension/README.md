# JuriScan — Extension Chrome

## Installation (mode développeur)

1. Ouvre Chrome → `chrome://extensions/`
2. Active **"Mode développeur"** (en haut à droite)
3. Clique **"Charger l'extension non empaquetée"**
4. Sélectionne le dossier `/extension`
5. L'extension apparaît dans ta barre Chrome

## Générer les icônes

1. Ouvre `generate_icons.html` dans Chrome
2. Les 3 icônes (16, 48, 128px) se téléchargent automatiquement
3. Déplace-les dans le dossier `icons/`

## Configuration

1. Clique sur l'icône JuriScan dans Chrome
2. Clique ⚙️ Paramètres
3. Entre ta clé API Anthropic (console.anthropic.com)
4. Enregistrer

## Utilisation

1. Ouvre une facture ou un devis dans Chrome (PDF, HTML, page web)
2. Clique sur JuriScan
3. Clique "Analyser cette page"
4. Résultat instantané avec les mentions manquantes
